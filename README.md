# Quick Sort (Divide and Conquer)

## Algorithm
Quick Sort selects a pivot element and partitions the array into two subarrays.

## Steps
1. Choose pivot
2. Partition array
3. Recursively sort left and right parts

## Time Complexity
- Best Case: O(n log n)
- Average Case: O(n log n)
- Worst Case: O(n^2)

## Space Complexity
- O(log n)

## Example
Input: 5 3 1 4 2  
Output: 1 2 3 4 5
